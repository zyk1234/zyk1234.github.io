redis 安装

  1. wget http://download.redis.io/releases/redis-3.0.6.tar.gz
  2. 安装gcc yum -y install gcc
  3. tar xzf redis-3.0.6.tar.gz
  4. 将2的解压文件夹移动到/usr/local/redis下
    mv redis-3.0.6 /usr/local/redis
  5. cd /usr/local/redis
  6. make && make install

reids windows

  1. 解压缩
  2. 配置环境变量
  3. 启动服务端 在cmd中 输入redis-server.exe
  4. 启动客户端 另起在cmd中，输入redis-cli.exe

ubuntu

  1. 切换到管理员 
    sudo -s
  2. 升级apt仓库源
    apt update
  3. 安装redis
    apt install redis-server
  4. 修改远程可访问
    vi /etc/redis/redis.conf
将结果里面的bing 127.0.0.1改成0.0.0.0
  5. 重启服务
    service redis-server restart

redis集群

参考:https://redis.io/topics/cluster-tutorial

集群规划：

生产下至少需要6台机器

  1. 创建集群目录

    mkdir redis_cluster

  2. 在集群目录下创建7000,7001,7002,7004,7005,7006

    cd redis_cluster
mkdir 7000 7001 7002

  3. 复制redis.conf到7000、7001、7002,7004,7005,7006

    cp /usr/local/redis/redis.conf /usr/local/redis_cluster/7000

  4. 修改7000下的redis配置文件

    vim redis.conf
bind 0.0.0.0 #绑定的ip地址
daemonize yes  #redis后台运行
pidfile /var/run/redis_7000.pid #pidfile对应7000，7001，7002,7004,7005,7006
port 7000 #端口号 7000，7001，7002,7004,7005,7006
cluster-enable yes #开启集群
cluster-config-file nodes_7000.conf #集群的配置 配置文件首次启动自动生成
cluster-node-timeout 5000 #请求超时，设置5秒就够了
appendonly  yes #aof日志开启  有需要就开启，它会每次写操作都记录一条日志

    将此文件复制到7001、7002,7004,7005,7006目录下

  5. 开启服务

    redis-server redis_cluster/7000/redis.conf
redis-server redis_cluster/7001/redis.conf
redis-server redis_cluster/7002/redis.conf
redis-server redis_cluster/7003/redis.conf
redis-server redis_cluster/7004/redis.conf
redis-server redis_cluster/7005/redis.conf

  6. 查看服务是否都启动起来了

    ss -nlt   #或者
ps -ef|grep redis

  7. 创建集群

    redis-cli --cluster create 172.168.20.206:7000 172.168.20.206:7001 172.168.20.206:7002 172.168.20.206:7003 172.168.20.206:7004 172.168.20.206:7005 --cluster-replicas 1

 --cluster-replicas 1  1  表示 自动为每一个master节点分配一个slave节点    上面有6个节点，程序会按照一定规则生成 3个master（主）3个slave(从)

  8. 检查集群

    redis-cli --cluster check 172.168.20.206:7000

  9. 测试

    redis-cli -c -p 7000
set hello world
#在各个节点上，查看
redis-cli -c -p 7002
get hello
