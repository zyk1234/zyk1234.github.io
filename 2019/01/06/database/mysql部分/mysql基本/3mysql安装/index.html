MySQL 安装

下载地址：https://dev.mysql.com/downloads/mysql/

windows安装

1. 下载完成后解压缩

2. 将解压缩后的bin目录配置环境变量

3. 在这个目下创建my.ini文件

其内容如下：

[mysql]
#设置3306端口
default-character-set=utf8
port = 3306


[mysqld]
#设置3306端口
port = 3306
# 设置mysql的安装目录
basedir=D:\mysql\mysql-8.0.13-winx64
# 设置mysql数据库的数据的存放目录
datadir=D:\mysql\mysql-8.0.13-winx64\data
# 允许最大连接数
max_connections=200
# 允许连接失败的次数。这是为了防止有人从该主机试图攻击数据库系统
max_connect_errors=10
# 服务端使用的字符集默认为8比特编码的latin1字符集
character-set-server=utf8
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB

4. 管理员身份打开cmd

5.切换到mysql解压缩的bin目录下

6.在cmd的bin目录下mysqld –install mysql –defaults-file=D:\mysql\mysql-8.0.13-winx64\my.ini

7.初始化mysql数据库，在cmd中继续输入 mysqld –initialize –user=root –console 拿到密码

8.net start mysql 启动服务

9.在cmd 中输入 mysql -u root -p 回车，将在第4步拿到的密码输入

10.设置密码

在mysql的客户端输入:

use mysql;
alter user 'root'@'localhost' identified with mysql_native_password by '123456';

flush privileges;

11.配置远程访问

mysql -uroot -p
use mysql;
update user set host = '%' where user ='root';
flush privileges;

Linux安装

centos7下的安装源码

从mysql官网下载 https://dev.mysql.com/downloads/mysql/ 页面中的linux版本下的64位的安装包，绿色安装即可

下载方式：
windows上下载下来，上传，或者linux上wget直接下载都可以

安装步骤：

  • 1.解压由1产生的tar文件
    tar zxvf mysql-8.0.13-el7-x86_64.tar.gz
  • 3.将2解压的文件夹移动到/usr/local/mysql目录下
    mv mysql-8.0.13-el7-x86_64 /usr/local/mysql

  • 4.添加mysql用户，linux是多用户的操作系统。专用的软件通常有专用的用户来管理

    useradd mysql -s /sbin/nologin

    /sbin/nologin 可以进行shell，但是不能用此用户登录

  • 5.初始化sql，初始化的时候，带上mysql用户即可

    ./mysqld -I -u mysql
    • 初始完成后，找 A temporary password is generated for root@localhost: xxxxxxxxxxx，其中 @localhost后面的即为初始化的初始密码，一定要将此密码拷贝下来，后续登录的时候要使用
  • 6.添加开机自启动
    将mysql 下的 support-files 文件夹下的mysql.server文件拷贝到 /etc/init.d/ 目录下
    cp /usr/local/mysql/support-files/mysql.server /etc/init.d/
    重命名
    mv /etc/init.d/mysql.server /etc/init.d/mysql

  • 7.添加软链接

    ln -s /usr/local/mysql/bin/mysql /usr/bin/mysql

  • 8.启动服务
    在启动服务之前，需要修改mysql的配置文件

    vi /etc/my.cnf

    将此文件下面的[mysqld_safe]下面的log-error和pid-file改成如下内容

    [mysqld_safe]
log-error=/var/log/mysql.log
pid-file=/var/run/mysql/mysql.pid

    同时将[mysqld]下面的socket=/var/lib/mysql/mysql.sock这行，拷贝到/etc/my.cnf.d/mysql-clients.cnf文件中的[mysql]下

    vi /etc/my.cnf.d/mysql-clients.cnf
[mysql]
socket=/var/lib/mysql/mysql.sock    
[mysql_upgrade]

    最后，启动服务器

    service mysql start

  • 9.修改密码和远程连接

    • 修改密码 

      mysql -uroot -p

      敲回车的时候，让输入密码，此处的密码即第5步产生的密码
      密码输入成功后，进入mysql的shell控制台
      在控制台输入：

      use mysql;
alter user 'root'@'localhost' identified with mysql_native_password by '123456';

flush privileges;

    • 密码修改后，修改远程连接

      use mysql;
update user set host='%' where user='root';

    • 重启mysql服务

      service mysql restart

  • 10.关闭防火墙

    systemctl stop firewalld.service #停止firewall
systemctl disable firewalld.service #禁止firewall开机启动

linux rpm包安装

下载地址

https://mirrors.163.com/mysql/Downloads/MySQL-8.0/mysql-8.0.13-1.el7.x86_64.rpm-bundle.tar

  1. 查看mariadb版本

    rpm -qa|grep mariadb

  2. 卸载mariadb

    rpm -e mariadb-libs-5.5.56-2.el7.x86_64 --nodeps

  3. 安装依赖

    yum install -y openssl-devel.x86_64 openssl.x86_64 libaio.x86_64 libaio-devel.x86_64 perl.x86_64 perl-devel.x86_64 perl-JSON.noarch autoconf wget net-tools

  4. 关闭防火墙和selinux

    systemctl stop firewalld.service
systemctl disable firewalld.service
#禁用selinux
setenforce 0 #临时禁用
vi /etc/selinux/config #长久禁用
将SELINUX=enforcing改为SELINUX=disabled

  5. 解压后按顺序安装

    rpm -ivh mysql-community-common-8.0.13-1.el7.x86_64.rpm
rpm -ivh mysql-community-libs-8.0.13-1.el7.x86_64.rpm
rpm -ivh mysql-community-client-8.0.13-1.el7.x86_64.rpm
rpm -ivh mysql-community-server-8.0.13-1.el7.x86_64.rpm

  1. 修改/etc/my.ini，在最后面加入

    skip-grant-tables

  2. 启动mysql

    systemctl  start  mysqld.service

  3. 进入

     mysql -uroot  #回车
flush privileges;
ALTER user 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123456';
  4. 重复6，将skip-grant-tables删除，然后重启mysql服务
